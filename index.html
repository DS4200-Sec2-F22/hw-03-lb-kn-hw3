hw-03: Interactive Webpage, and Marks & Channels

Marks

  • Areas
  • Lines

Channels

  • Color
    • Color hue represents a categorical attribute: party affiliation
    • This is a good mapping because it clearly shows what category each state is in using colors that people already associate with these categories
  • Length or Horizontal Position
    • Position on a common scale/length represents a quantitative attribute: projected electoral votes for each party
    • This is a good mapping because position on a common scale is the most effective magnitude channel or ...

Colormap(s)

  • Categorical
    • This is a good mapping because it clearly shows what category each state is in using colors that people already associate with these categories

Marks

  • Points

Channels

  • Color
    • Color saturation represents a quantitative attribute: scoring value
    • Color hue represents a categorical attribute: typical vs atypical scoring value
    • This is a poor mapping because it is difficult to interpret what the significance is of the saturation
  • Position
    • Both horizontal and vertical position represent quantitative attributes: exit velocity & launch angle (respectively)
    • This is a good mapping because it shows the magnitude in both measures and the relationship between these measures clearly and in a way that is easy to read and interpret

Colormap(s)

  • Sequential
    • This is a poor mapping choice because it is nit very clear what value this information provides nor really what it means in he context of the graph
  • Categorical
    • This is an ok mapping because it does communicate the categories the points are in and helps to group points together
