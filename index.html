hw-03: Interactive Webpage, and Marks & Channels

Marks

  • Areas
  • Lines

Channels

  • Color
    • Color hue represents a categorical attribute: party affiliation
    • This is a good mapping because it clearly shows what category each state is in using colors that people already associate with these categories.
  • Length
    • Length represents a quantitative attribute: projected electoral votes for each party
    • This is a good mapping because it's very clear what is trying to be communicated, where longer length means greater number of votes.

Colormap(s)

  • Categorical
    • This is a good mapping because it clearly shows what category each state is in using colors that people already associate with these categories.

Marks

  • Points

Channels

  • Color
    • Color saturation represents a quantitative attribute: scoring value
    • Color hue represents a categorical attribute: above average vs below average scoring value
    • This is a poor mapping because the choices of hue, purple and orange, seem unrelated and don't provide any additional meaning (ex. positive or negative connotation). Also, the use of white for the average value could be easily mistaken for missing data since it is similar to the background color.
  • Position
    • Both horizontal and vertical position represent quantitative attributes: exit velocity & launch angle (respectively)
    • This is a good mapping because it shows the magnitude in both measures and the relationship between these measures clearly and in a way that is easy to read and interpret.

Colormap(s)

  • Sequential
    • This is an good mapping choice because it is the best way to show that the data (score) lies along a continuum, however it was poorly excecuted since average is difficult to see and the different hues get in the way.
  • Categorical
    • This is a poor mapping because while it does helps to group points together, it seems to be secondary to the point of the visualization, which is to show scores along a single continuum.
